# Headless CMS dengan WordPress - Slide untuk Virtual Summit NiagaHoster 2020

## Notes
**Slides are using Indonesian** 

## Installation

```bash
$ git clone git@github.com:adriantirusli/slide-headless-cms.git
```

## Usage
```bash
# To develop & write
$ yarn develop

# To build
$ yarn build
```
